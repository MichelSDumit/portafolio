Práctica 7: Brazo Robotico Bluetooth

Michel Sánchez Dumit
Ing. Mecatrónica
Giovanna González Martínez
Ing. Mecatrónica
Fecha: Marzo 21, 2025
Arduino UNO Electrónica Robótica Bluetooth

- Resumen -

Esta práctica se enfocó en la implementación de un sistema de control mediante Bluetooth para el brazo robótico desarrollado en la práctica 4. Utilizando la plataforma MIT App Inventor, se diseñó y desarrolló una aplicación móvil que permite controlar las posiciones de los servomotores mediante diales. Esta interfaz facilita el ajuste preciso de las articulaciones del brazo de forma remota, mejorando la experiencia de usuario y expandiendo las capacidades del sistema robótico.

- Introducción -

Siguiendo los conocimientos adquiridos en la práctica 6, reemplazamos el sistema de control basado en potenciómetros por una aplicación móvil y el módulo Bluetooth HC05. La interfaz de la aplicación está compuesta por cuatro diales, cada uno asignado a un servomotor específico. Al ajustar cualquiera de los diales, se envía un valor que identifica el servomotor cuya posición se debe modificar, seguido de un valor entre 0 y 180 que indica la posición deseada para ese servo.

Una vez que los datos son recibidos por el módulo HC05, el Arduino procesa la información y ajusta los servomotores a las posiciones solicitadas por la aplicación, logrando así un control preciso y remoto de los movimientos del brazo robótico.

- Materiales -

Placa Arduino UNO.

Módulo Bluetooth HC05.

Cable USB para la conexión con el ordenador.

Servomotores (5 unidades).

Jumpers.

Fuente de alimentación externa.

- Desarrollo -

La aplicación y el código de Arduino fueron desarrollados de manera simultánea para asegurar que los datos enviados desde el celular fueran procesados correctamente. Esta sincronización nos permitió ajustar ambos sistemas y garantizar su correcta interacción.

Diseño/h1>

El diseño de la interfaz de la aplicación es sencillo, compuesto únicamente por tres botones y cinco diales. Los botones cumplen funciones específicas: conexión, desconexión y salida de la aplicación. Por otro lado, los diales están destinados a controlar el movimiento del brazo robótico, permitiendo ajustar la posición de cada servomotor de manera precisa y directa.

Construcción

La construcción de los circuitos consistió principalmente en reemplazar el sistema de control basado en potenciómetros por el módulo HC05. Esto implicó desinstalar los componentes anteriores y realizar las conexiones necesarias para integrar el módulo Bluetooth al sistema, permitiendo así la comunicación inalámbrica entre la aplicación y el Arduino.

- Resultados -

El brazo robótico es ahora capaz de cumplir con todas las funciones de movimiento esperadas, y ha demostrado una mayor precisión y un mejor tiempo de respuesta en comparación con el sistema anterior basado en potenciómetros.

- Conclusiones -

Se cumplió exitosamente la meta de la práctica, logrando integrar un sistema de control inalámbrico en el brazo robótico. Esta implementación permitió mejorar significativamente la precisión y eficiencia del sistema, demostrando cómo las conexiones inalámbricas pueden optimizar el control de maquinaria y facilitar su operación a distancia.
